RBAC in DBMS
> RBAC : Role-Based Access Control, 역할 기반 접근 제어
> 사용자,역할,권한,할당 등의 개념을 가지고 있음
> 각 역할에 적절한 권한을 부여하고 사용자에게 하나 이상의 역할을 부여함.
> DBMS : Database Management System, DB 관리 시스템


ex)
-- 역할 생성 > manager 역할을 생성하고 해당 역할에 employees 테이블에 해당 쿼리 권한을 부여함
CREATE ROLE manager;
GRANT SELECT, INSERT ON employees TO manager;

-- 사용자에게 역할 할당 > 해당하는 사용자에게 manager 역할을 부여함 > alice는 employees 테이블에 SELECT,INSERT 권한을 가짐
CREATE USER alice WITH PASSWORD 'securepass';
GRANT manager TO alice;
